Features

🔒

User Auth

Secure registration and login, SQLite-backed sessions. Your data stays on-device.

🤖

AI Chat

Conversational AI powered by local Ollama models via 4-node cluster proxy. No OpenAI. No data leaving your network.

🧠

Weaviate RAG

Semantic search across 210+ knowledge objects. Responses grounded in your personal knowledge base.

📚

Messenger Storyteller

A narrative wellness engine built into the messenger. Stories unfold through conversation over time.

🔒

Privacy Commands

OPTOUT, ANON, FORGET — users control their data at any time with plain-text commands.

🏆

Leaderboard

Community wellness leaderboard. Friendly accountability, no surveillance.

🧑

Discord Integration

Connected to the ShaneBrain Discord bot. Notifications, alerts, and morning briefings.

📈

SecurityLog + PrivacyAudit

Every gateway event logged to Weaviate. Full audit trail without third-party analytics.

🌐

Public via Tailscale Funnel

Exposed securely over HTTPS at the public URL without opening firewall ports.
